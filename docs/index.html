Gemini MCP Server

Claude Desktop에서 Gemini AI의 강력한 기능을 활용하세요. 실시간 출력, 파일 분석, 샌드박스 모드 등 다양한 기능을 제공합니다.

🚀 빠른 시작

아래 단계를 따라 Gemini MCP 서버를 설치하고 설정하세요.

1단계: 프로젝트 다운로드

git clone https://github.com/hohollala/Gemini_Mcp.git

2단계: 프로젝트 폴더로 이동

cd Gemini_Mcp

3단계: 의존성 설치

npm install

4단계: 프로젝트 빌드

npm run build

5단계: 명령어 생성

npm run generate-commands

6단계: 글로벌 설치

npm install .

7단계: MCP 서버 등록

Windows (PowerShell)

claude mcp add gemini-mcp -s user -- node $PWD/dist/index.js

Windows (CMD)

claude mcp add gemini-mcp -s user -- node [설치경로]/dist/index.js

macOS/Linux

claude mcp add gemini-mcp -s user -- node $(pwd)/dist/index.js

Windows (PowerShell)

claude mcp add gemini-mcp -s workspace -- node $PWD/dist/index.js

Windows (CMD)

claude mcp add gemini-mcp -s workspace -- node [설치경로]/dist/index.js

macOS/Linux

claude mcp add gemini-mcp -s workspace -- node $(pwd)/dist/index.js

✨ 주요 기능

🤖 Gemini CLI 연동

Google의 Gemini AI와 완벽하게 연동되어 강력한 AI 기능을 제공합니다.

📊 실시간 출력

명령어 실행 결과를 실시간으로 확인할 수 있어 즉시 피드백을 받을 수 있습니다.

📁 파일 분석

@filename 구문을 사용하여 파일을 직접 분석하고 코드 리뷰를 받을 수 있습니다.

🏖️ 샌드박스 모드

안전한 환경에서 코드를 실행하고 테스트할 수 있습니다.

🎯 모델 선택

다양한 Gemini 모델 중에서 선택하여 사용할 수 있습니다.

🌍 글로벌 설치

시스템 전체에서 사용할 수 있도록 글로벌 설치를 지원합니다.

💡 사용 예시

기본 사용법

간단한 질문

/gc-ask "JavaScript에서 배열을 정렬하는 방법을 알려줘"

파일 분석

/gc-ask "이 파일의 의존성을 분석해줘: @package.json"

코드 리뷰

/gc-ask "이 코드를 리뷰하고 개선점을 제안해줘: @src/index.ts"

문서 생성

/gc-ask "이 파일을 기반으로 API 문서를 생성해줘: @README.md"

고급 사용법

샌드박스 모드

/gc-ask "샌드박스 모드에서 Python으로 간단한 웹 서버를 만들어줘" --sandbox

모델 선택

/gc-ask "복잡한 알고리즘을 설명해줘" --model "gemini-2.0-flash-exp"

유틸리티 명령어

# 서버 상태 확인 /gc-ping "테스트 메시지" # 도움말 보기 /gc-help

🔧 문제 해결

Gemini CLI 설치 문제

⚠️ 주의: Gemini CLI가 설치되어 있지 않으면 MCP 서버가 작동하지 않습니다.
npm install -g @google/generative-ai

MCP 서버 연결 실패

# MCP 서버 상태 확인 claude mcp list # 서버 재등록 claude mcp remove gemini-mcp claude mcp add gemini-mcp -s user -- node [경로]/dist/index.js

Windows 특별 문제

# PowerShell에서 실행 정책 변경 Set-ExecutionPolicy -ExecutionPolicy RemoteSigned -Scope CurrentUser # CMD에서 절대 경로 사용 claude mcp add gemini-mcp -s user -- "C:\Users\%USERNAME%\AppData\Roaming\npm\gemini-mcp.cmd"

npm install 에러

# 캐시 정리 npm cache clean --force # node_modules 삭제 후 재설치 rm -rf node_modules package-lock.json npm install
